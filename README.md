# Rexdump
### Minimalistic hexdump implementation in Rust

![](imgs/preview.png)

# Installation
    git clone https://github.com/skynse/rexdump
    cd rexdump
    cargo install --path .

# Usage

    Rexdump 0.1.1
    minimalist hexdump implementation

    USAGE:
        rexdump [OPTIONS]

    FLAGS:
        -h, --help       Prints help information
        -V, --version    Prints version information

    OPTIONS:
        -i <FILE>        Input file
